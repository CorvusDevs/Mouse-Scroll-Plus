# Mouse Scroll Plus

Smooth, trackpad-like scrolling for any mouse on macOS. Turn each wheel notch into a fluid glide, reverse direction, tune speed and momentum, remap extra buttons, and set per-app rules. Private and lightweight, everything runs on your Mac.

- **Website:** https://corvusdevs.github.io/Mouse-Scroll-Plus/
- **Support:** https://corvusdevs.github.io/Mouse-Scroll-Plus/support.html
- **Privacy:** https://corvusdevs.github.io/Mouse-Scroll-Plus/privacy.html

For macOS 13 and later. Made by [CorvusDevs](https://corvusdevs.github.io/).

This repository hosts the public website (GitHub Pages). The app source lives in a private mirror.
